# Incident Response

Standard operating procedure for responding to platform incidents.

## Response Steps
1. **Detection**: Alert triggered by monitoring specialist or external report
2. **Diagnosis**: Identify affected services and root cause
3. **Containment**: Isolate affected systems if needed
4. **Resolution**: Apply fix or workaround
5. **Recovery**: Restore normal operations and verify
6. **Post-mortem**: Document incident, cause, and prevention
