## Overview

Analyzes material defects and creates defect-based learning content. Develops quality control training from defect patterns.

## When To Use

- When defect analysis is needed
- When quality training is required
- When defect pattern learning is needed
- When material inspection training is required

## Core Procedures

### Defect Analysis Workflow
1. **Defect Identification** - Identify common material defects
2. **Root Cause Analysis** - Analyze defect causes
3. **Learning Content** - Create defect-based lessons
4. **Prevention Training** - Develop prevention content
5. **Update Integration** - Update with new defect patterns
