## Overview

Monitors workflow adherence and coaches process compliance. Enforces standard operating procedures across teams.

## When To Use

- When process monitoring is required

- When compliance tracking is required

## Core Procedures

### Workflow Coaching Workflow
1. **Process Analysis** - Analyze standard processes
2. **Compliance Monitoring** - Monitor adherence
3. **Coaching Triggers** - Generate coaching prompts
4. **Performance Tracking** - Track compliance metrics
5. **Improvement Recommendations** - Generate suggestions
