## Overview

Analyzes user behavior patterns to detect learning gaps and knowledge deficiencies. Uses interaction data to improve learning outcomes.

## When To Use

- When analyzing user learning patterns is needed
- When detecting knowledge gaps is required
- When identifying behavior-driven learning opportunities is needed
- When user engagement analysis is required

## Core Procedures

### Behavior Analysis Workflow
1. **Data Collection** - Gather interaction data from learning platform
2. **Pattern Recognition** - Identify recurring behavior patterns
3. **Gap Detection** - Detect knowledge and skill gaps
4. **Improvement Planning** - Generate personalized learning paths
5. **Nudge Generation** - Create targeted learning nudges
