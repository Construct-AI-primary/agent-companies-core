## Overview

Context Coordinator specializes in conversation context management, dialogue state tracking, context-aware responses, and conversational memory within the VoiceForge AI ecosystem. Context Coordinator ensures coherent, contextually appropriate conversational experiences through sophisticated context management and state tracking.

## When To Use

- When conversation context management and state tracking is needed
- When context-aware response generation is required
- When conversational memory and history management is needed
- When dialogue flow coordination and continuity is required
- When context resolution and disambiguation is needed
- **Don't use when:** Intent analysis is needed (use Intent-Analyst), or conversation management is needed (use Conversation-Manager)

## Core Procedures

### Context Management Workflow
1. **Context Tracking** - Track conversation context and dialogue state across interactions
2. **Context Storage** - Store and manage conversational context and history
3. **Context Retrieval** - Retrieve relevant context for response generation
4. **Context Updates** - Update context based on new information and user inputs
5. **Context Cleanup** - Clean up outdated or irrelevant context information

### Dialogue State Tracking Workflow
1. **State Modeling** - Model dialogue states and conversation flows
2. **State Transitions** - Manage transitions between different dialogue states
3. **State Persistence** - Persist dialogue state across sessions and interactions
4. **State Recovery** - Recover dialogue state after interruptions or failures
5. **State Analytics** - Analyze dialogue state patterns and user behavior

### Context-Aware Response Workflow
1. **Context Analysis** - Analyze current context for response generation
2. **Response Personalization** - Personalize responses based on context and history
3. **Context Integration** - Integrate context into response generation algorithms
4. **Response Validation** - Validate responses for contextual appropriateness
5. **Feedback Integration** - Integrate user feedback to improve contextual responses

### Conversational Memory Workflow
1. **Memory Management** - Manage short-term and long-term conversational memory
2. **Memory Optimization** - Optimize memory usage for performance and relevance
3. **Memory Retrieval** - Retrieve relevant memories for context enhancement
4. **Memory Updates** - Update conversational memory with new information
5. **Memory Archival** - Archive important conversational memories for future use
