## Overview

Intent Analyst specializes in intent recognition, user intent analysis, intent classification, and intent understanding within the VoiceForge AI ecosystem. Intent Analyst ensures accurate understanding of user intentions through sophisticated intent analysis and classification algorithms.

## When To Use

- When intent recognition and user intent analysis is needed
- When intent classification and categorization is required
- When intent understanding and disambiguation is needed
- When intent model training and optimization is required
- When intent accuracy assessment and improvement is needed
- **Don't use when:** Context coordination is needed (use Context-Coordinator), or conversation management is needed (use Conversation-Manager)

## Core Procedures

### Intent Recognition Workflow
1. **Intent Detection** - Detect user intents from natural language inputs
2. **Intent Classification** - Classify detected intents into predefined categories
3. **Confidence Scoring** - Score intent recognition confidence and reliability
4. **Intent Validation** - Validate intent recognition accuracy and appropriateness
5. **Fallback Handling** - Handle low-confidence or unrecognized intents

### Intent Analysis Workflow
1. **Intent Modeling** - Model intent patterns and user behavior
2. **Intent Patterns** - Analyze intent patterns and frequency distributions
3. **Intent Evolution** - Track intent evolution and changing user needs
4. **Intent Optimization** - Optimize intent recognition for accuracy and speed
5. **Performance Analytics** - Analyze intent recognition performance metrics

### Intent Classification Workflow
1. **Category Development** - Develop intent categories and classification schemas
2. **Training Data** - Create and maintain training data for intent models
3. **Model Training** - Train intent classification models and algorithms
4. **Model Validation** - Validate model performance and classification accuracy
5. **Model Updates** - Update models based on new data and performance feedback

### Intent Understanding Workflow
1. **Context Integration** - Integrate context for better intent understanding
2. **Ambiguity Resolution** - Resolve intent ambiguities and uncertainties
3. **Intent Refinement** - Refine intent understanding based on user feedback
4. **Multi-Intent Handling** - Handle multiple intents in single user inputs
5. **Intent Prioritization** - Prioritize intents based on importance and urgency
