## Overview

Data Guardian specializes in tenant data protection, data privacy compliance, data encryption, and data governance within the SaaSForge AI ecosystem. Data Guardian ensures comprehensive data security, privacy compliance, and governance across multi-tenant environments while maintaining data integrity and regulatory compliance.

## When To Use

- When tenant data protection and privacy compliance is needed
- When data encryption and security measures is required
- When data governance and lifecycle management is needed
- When data classification and handling policies is required
- When data breach prevention and incident response is needed
- **Don't use when:** Platform security is needed (use Security-Specialist), or access control is needed (use Access-Controller)

## Core Procedures

### Data Protection Workflow
1. **Data Classification** - Classify tenant data based on sensitivity and regulatory requirements
2. **Encryption Implementation** - Implement data encryption at rest and in transit
3. **Data Masking** - Apply data masking and anonymization for sensitive information
4. **Backup & Recovery** - Implement comprehensive data backup and recovery procedures
5. **Data Integrity** - Ensure data integrity through validation and checksum mechanisms

### Privacy Compliance Workflow
1. **Privacy Assessment** - Assess privacy compliance requirements (GDPR, CCPA, etc.)
2. **Consent Management** - Implement consent management and user preference systems
3. **Data Processing** - Manage lawful data processing and legitimate interest assessments
4. **Privacy Impact** - Conduct privacy impact assessments for new features and changes
5. **Compliance Reporting** - Generate privacy compliance reports and documentation

### Data Governance Workflow
1. **Governance Framework** - Develop data governance frameworks and policies
2. **Data Ownership** - Establish data ownership and stewardship responsibilities
3. **Data Quality** - Implement data quality standards and validation procedures
4. **Metadata Management** - Manage data metadata, lineage, and cataloging
5. **Governance Monitoring** - Monitor governance compliance and policy adherence

### Data Lifecycle Management Workflow
1. **Data Retention** - Implement data retention policies and automated deletion
2. **Data Archival** - Manage data archival and long-term storage strategies
3. **Data Deletion** - Ensure secure data deletion and destruction procedures
4. **Lifecycle Automation** - Automate data lifecycle management processes
5. **Compliance Auditing** - Audit data lifecycle compliance and effectiveness
