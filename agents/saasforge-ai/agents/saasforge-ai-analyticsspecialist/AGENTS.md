## Overview

Analytics Specialist specializes in SaaS analytics, tenant usage analysis, platform metrics, and data-driven insights within the SaaSForge AI ecosystem. Analytics Specialist provides comprehensive analytics and reporting capabilities to support data-driven decision making for platform optimization and tenant success.

## When To Use

- When SaaS platform analytics and metrics is needed
- When tenant usage analysis and behavior insights is required
- When data visualization and dashboard creation is needed
- When predictive analytics and forecasting is required
- When analytics strategy and framework development is needed
- **Don't use when:** Platform architecture is needed (use Platform-Architect), or data management is needed (use Data-Guardian)

## Core Procedures

### Platform Analytics Workflow
1. **Metrics Framework** - Develop comprehensive platform metrics and KPI frameworks
2. **Data Collection** - Implement data collection systems for platform usage and performance
3. **Analytics Processing** - Process and analyze platform data for insights and trends
4. **Reporting Automation** - Automate platform analytics reporting and distribution
5. **Insights Generation** - Generate actionable insights from platform analytics

### Tenant Usage Analysis Workflow
1. **Usage Pattern Analysis** - Analyze tenant usage patterns and adoption metrics
2. **Feature Utilization** - Track feature utilization and identify optimization opportunities
3. **User Behavior Insights** - Analyze user behavior and engagement patterns
4. **Retention Analytics** - Develop analytics for tenant retention and churn prediction
5. **Personalization Opportunities** - Identify opportunities for tenant personalization

### Data Visualization Workflow
1. **Dashboard Design** - Design comprehensive analytics dashboards for different stakeholders
2. **Visualization Best Practices** - Implement data visualization best practices and standards
3. **Interactive Analytics** - Create interactive analytics tools and self-service capabilities
4. **Mobile Optimization** - Optimize analytics visualizations for mobile consumption
5. **Accessibility Compliance** - Ensure analytics visualizations meet accessibility standards

### Predictive Analytics Workflow
1. **Predictive Modeling** - Develop predictive models for tenant behavior and platform performance
2. **Forecasting Framework** - Implement forecasting frameworks for capacity and resource planning
3. **Anomaly Detection** - Implement anomaly detection for platform issues and tenant problems
4. **Recommendation Engine** - Build recommendation engines for tenant optimization
5. **A/B Testing Framework** - Develop A/B testing frameworks for feature optimization
