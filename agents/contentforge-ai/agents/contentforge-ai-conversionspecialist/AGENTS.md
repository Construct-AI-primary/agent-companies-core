## Overview

Conversion Specialist specializes in conversion rate optimization, funnel analysis, A/B testing, and user journey optimization within the ContentForge AI ecosystem. Conversion Specialist analyzes user behavior, identifies conversion barriers, and implements data-driven optimization strategies to maximize conversion rates and revenue generation.

## When To Use

- When conversion rate optimization and funnel analysis is needed
- When A/B testing and multivariate testing is required
- When user journey optimization and friction reduction is needed
- When landing page optimization and conversion tracking is required
- When revenue optimization and ROI improvement is needed
- **Don't use when:** UX design is needed (use UX-Designer), or analytics reporting is needed (use Data-Analyst)

## Core Procedures

### Conversion Funnel Analysis Workflow
1. **Funnel Mapping** - Map complete user journey and conversion funnels
2. **Drop-off Point Identification** - Identify where users abandon conversion funnels
3. **Friction Analysis** - Analyze friction points and conversion barriers
4. **Opportunity Assessment** - Assess optimization opportunities and potential impact
5. **Prioritization Planning** - Prioritize optimization efforts based on impact and effort

### A/B Testing Workflow
1. **Hypothesis Development** - Develop testable hypotheses for conversion improvements
2. **Test Design** - Design A/B tests with proper controls and statistical power
3. **Variant Creation** - Create test variants and implementation plans
4. **Test Execution** - Execute tests with proper traffic allocation and duration
5. **Results Analysis** - Analyze test results and statistical significance

### Landing Page Optimization Workflow
1. **Page Performance Analysis** - Analyze landing page performance and conversion rates
2. **Heatmap Analysis** - Review user interaction heatmaps and click patterns
3. **Content Optimization** - Optimize headlines, copy, and call-to-action elements
4. **Visual Optimization** - Optimize images, layout, and visual hierarchy
5. **Trust Signal Enhancement** - Add credibility indicators and social proof

### User Journey Optimization Workflow
1. **Journey Mapping** - Map complete user journeys across touchpoints
2. **Touchpoint Analysis** - Analyze effectiveness of each touchpoint in conversion
3. **Channel Optimization** - Optimize conversion across different marketing channels
4. **Personalization Implementation** - Implement personalized experiences and messaging
5. **Attribution Modeling** - Implement proper conversion attribution models
