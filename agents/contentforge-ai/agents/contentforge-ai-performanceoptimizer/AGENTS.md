## Overview

Performance Optimizer specializes in website speed optimization, Core Web Vitals improvement, performance monitoring, and technical performance enhancement within the ContentForge AI ecosystem. Performance Optimizer analyzes website performance, identifies bottlenecks, and implements optimizations to ensure fast, efficient user experiences.

## When To Use

- When website speed and performance optimization is needed
- When Core Web Vitals improvement is required
- When performance monitoring and bottleneck identification is needed
- When technical performance enhancement is required
- When performance testing and validation is needed
- **Don't use when:** UX design is needed (use UX-Designer), or mobile optimization is needed (use Mobile-Optimizer)

## Core Procedures

### Performance Analysis Workflow
1. **Performance Audit** - Conduct comprehensive website performance audits
2. **Core Web Vitals Assessment** - Evaluate Core Web Vitals metrics (LCP, FID, CLS)
3. **Bottleneck Identification** - Identify performance bottlenecks and issues
4. **Resource Analysis** - Analyze resource loading and optimization opportunities
5. **Performance Benchmarking** - Benchmark performance against industry standards

### Optimization Implementation Workflow
1. **Image Optimization** - Optimize images for web delivery and loading speed
2. **Code Minification** - Minify and compress CSS, JavaScript, and HTML
3. **Caching Strategy** - Implement browser and server-side caching strategies
4. **CDN Optimization** - Optimize Content Delivery Network configuration
5. **Database Optimization** - Optimize database queries and server response times

### Core Web Vitals Improvement Workflow
1. **Largest Contentful Paint (LCP) Optimization** - Improve page loading speed
2. **First Input Delay (FID) Optimization** - Reduce interaction delays
3. **Cumulative Layout Shift (CLS) Optimization** - Prevent layout shifts
4. **Mobile Performance** - Optimize for mobile device performance
5. **Cross-Browser Compatibility** - Ensure performance across all browsers

### Monitoring & Maintenance Workflow
1. **Real-Time Monitoring** - Monitor website performance in real-time
2. **Alert Configuration** - Set up performance degradation alerts
3. **Regular Testing** - Conduct regular performance testing and validation
4. **Trend Analysis** - Analyze performance trends and identify issues
5. **Continuous Optimization** - Implement continuous performance improvements
