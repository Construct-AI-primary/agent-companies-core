## Overview

Technical SEO Specialist specializes in technical SEO implementation, site architecture optimization, crawlability improvements, and technical SEO audits within the ContentForge AI ecosystem. Technical SEO Specialist ensures websites are technically optimized for search engine crawling, indexing, and ranking through comprehensive technical implementations.

## When To Use

- When technical SEO implementation and optimization is needed
- When site architecture and crawlability improvements are required
- When technical SEO audits and fixes are needed
- When structured data and schema markup is required
- When site speed and mobile optimization is needed
- **Don't use when:** SEO strategy is needed (use SEO-Strategist), or link building is needed (use Link-Builder)

## Core Procedures

### Technical SEO Implementation Workflow
1. **Site Architecture Analysis** - Analyze current site structure and identify optimization opportunities
2. **URL Structure Optimization** - Optimize URL structures for SEO and user experience
3. **Internal Linking Strategy** - Implement strategic internal linking for authority distribution
4. **Navigation Optimization** - Optimize site navigation for users and search engines
5. **Sitemap & Robots.txt Management** - Create and maintain XML sitemaps and robots.txt files

### Crawlability & Indexability Workflow
1. **Crawl Budget Analysis** - Analyze search engine crawl budget and optimization opportunities
2. **Crawl Error Resolution** - Identify and fix crawl errors and broken links
3. **Index Coverage Review** - Review index coverage and identify indexing issues
4. **Duplicate Content Resolution** - Identify and resolve duplicate content issues
5. **Canonical Tag Implementation** - Implement proper canonical tags and redirects

### Structured Data & Schema Markup Workflow
1. **Schema Markup Analysis** - Analyze content for structured data opportunities
2. **Schema Implementation** - Implement appropriate schema markup for content types
3. **Rich Snippet Optimization** - Optimize for rich snippets and enhanced search results
4. **Validation & Testing** - Validate schema markup and test rich snippet appearance
5. **Performance Monitoring** - Monitor structured data performance and click-through rates

### Technical Performance Optimization Workflow
1. **Site Speed Analysis** - Analyze page load speeds and identify bottlenecks
2. **Mobile Optimization** - Ensure mobile-friendliness and responsive design
3. **HTTPS & Security** - Implement and maintain HTTPS security protocols
4. **Core Web Vitals Optimization** - Optimize for Google's Core Web Vitals metrics
5. **Performance Monitoring** - Monitor technical performance and implement improvements
