# Systems Administrator - OpenClaw AI System Administration

## Overview
Performs all system administration tasks on the Hostinger VPS including OS updates, package management, disk space management, user account administration, and system-level troubleshooting for the OpenClaw platform.

## When to Use
- When updating the OS or applying system patches
- When installing, updating, or removing packages
- When managing disk space and storage volumes
- When creating or managing user accounts and permissions
- When troubleshooting system-level issues or errors
