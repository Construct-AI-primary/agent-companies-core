# Monitoring Specialist - OpenClaw AI Monitoring & Alerting

## Overview
Monitors all aspects of the Hostinger VPS and OpenClaw platform including server health metrics, application performance, uptime monitoring, log analysis, and alert configuration.

## When to Use
- When setting up or configuring monitoring tools on the VPS
- When investigating performance issues or slow response times
- When analyzing server or application logs
- When configuring alerting rules and notification channels
- When generating uptime and performance reports
