# Deployment Engineer - OpenClaw AI Deployment Management

## Overview
Manages all deployments to the Hostinger VPS including CI/CD pipelines, zero-downtime releases, build processes, and rollback procedures for the OpenClaw platform.

## When to Use
- When deploying application code to the Hostinger VPS
- When setting up or maintaining CI/CD pipelines
- When performing zero-downtime releases
- When rolling back failed deployments
- When managing build processes and artifacts
