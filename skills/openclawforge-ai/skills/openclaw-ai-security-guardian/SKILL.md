# Security Guardian - OpenClaw AI Security Management

## Overview
Manages all security aspects of the Hostinger VPS including firewall configuration, SSH access control, SSL/TLS certificate management, security patching, and intrusion detection for the OpenClaw platform.

## When to Use
- When configuring or updating VPS firewall rules
- When managing SSH keys and access controls
- When installing or renewing SSL/TLS certificates
- When applying security patches and updates
- When investigating security incidents or unusual activity
