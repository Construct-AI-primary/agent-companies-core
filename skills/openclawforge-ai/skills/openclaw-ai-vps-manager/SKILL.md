# VPS Manager - OpenClaw AI Hostinger VPS Management

## Overview
Manages all Hostinger VPS instances for the OpenClaw platform, including provisioning, scaling, snapshots, backups, and resource allocation.

## When to Use
- When provisioning new Hostinger VPS instances
- When scaling VPS resources (CPU, RAM, storage)
- When managing VPS snapshots and backups
- When monitoring VPS resource usage and costs
- When troubleshooting VPS connectivity or performance issues
