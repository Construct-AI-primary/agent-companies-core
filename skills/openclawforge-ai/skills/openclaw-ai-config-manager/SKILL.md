# Config Manager - OpenClaw AI Configuration Management

## Overview
Manages all configuration aspects of the OpenClaw platform including environment variables, .env files, application config files, deployment configurations, and secrets management across the Hostinger VPS.

## When to Use
- When managing environment variables for different environments
- When updating configuration files for applications or services
- When managing secrets and credentials securely
- When setting up configuration for new deployments
- When auditing or documenting configuration settings
